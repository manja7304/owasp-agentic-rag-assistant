# Security Policy

## Responsible Use

**OWASP Agentic RAG Assistant** is for educational and demonstration purposes only.

- All demo data is **synthetic** — no real PII, credentials, or production systems.
- Do **not** use this software against systems without explicit written authorization.
- No live exploitation, scanning, or cloud account access is performed by default.

## Reporting

Report security issues via GitHub Issues on this repository (no sensitive details in public issues).
